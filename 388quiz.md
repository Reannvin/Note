## 1.==与equals的区别

### 答案：

==对比的是栈中的值，基本数据类型是变量值，引用类型是堆中内存变量的地址。
equals通常会重写

    String str1 = "Hello";//在堆中的常量池分配内存
    String str2 = new String("Hello");//堆中直接分配内存
    String str3 = str2;//通过引用地址赋值

对于“==”来说，str1 ≠ str2 = str3（只比较地址）
但是equals来说，三个字符串复写后内容都是一样的。

### 自己的理解：
==：栈中比数值，堆中比地址。
equals只比较字符串，不比较地址啥的。
